## 快速开始

### 方式一：使用启动脚本（推荐）

Windows 用户：
  双击运行 start.bat

macOS/Linux 用户：
  chmod +x start.sh
  ./start.sh

### 方式二：手动启动

1. 安装依赖
   npm install
   pip install -r python_backend/requirements.txt

2. 下载模型（首次使用）
   python python_backend/download_model.py
   
   或使用魔搭命令：
   modelscope download --model HuggingFaceTB/SmolVLM-500M-Instruct

3. 启动应用
   npm start


## 功能说明

✨ 主要功能：
- 智能对话：与 AI 虚拟人物自然交流
- 图片理解：支持上传图片进行多模态对话
- 本地运行：完全离线，保护隐私
- 现代 UI：精美的聊天界面设计

🎮 操作指南：
- 发送消息：输入文字后按 Enter 发送
- 换行：按 Shift + Enter
- 上传图片：点击左侧图片按钮
- 新对话：点击侧边栏"新对话"按钮
- 查看状态：左侧显示连接状态和模型信息


## 系统要求

必需：
- Python 3.8 或更高版本
- Node.js 16 或更高版本
- 至少 4GB 内存
- 约 2GB 磁盘空间（模型文件）

推荐：
- 8GB 或更多内存
- NVIDIA GPU（自动使用 CUDA 加速）


## 故障排除

问题：启动后一直显示"连接中"
解决：
1. 检查 Python 后端是否启动（查看控制台日志）
2. 确认 5000 端口未被占用
3. 检查防火墙设置

问题：模型加载失败
解决：
1. 检查网络连接
2. 手动下载模型：python python_backend/download_model.py
3. 查看 Python 日志了解具体错误

问题：CPU 占用过高
解决：
1. 这是正常现象（AI 推理需要计算）
2. 有 GPU 的话会自动使用 GPU 加速
3. 可以关闭不使用的其他程序
